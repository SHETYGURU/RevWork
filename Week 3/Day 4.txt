slowly changing dimension(scd) -technique used to tarck changes in dimension coloumn

3 techniques
scd1- if any Dimensional attributs change update old value with new value(done if we dont need history)
scd2- insert new record with new changes with new surrogate key
scd3- add new column to dimension table

types of olap
rolap -  relational online analytical processing database [data stored in relational db using star ,snowflake,gal...]
        [supports/stores huge amount of data][query performance is slow]

molap -  multi-dimensional ---- [data will be stored in multi dimensional cubes using multi dimensional arrys]
        [uses indexing to fetch data][ex:-essbase,cognos transformer,ssas][faster than rolap][cant store huge amount of data]

holap -  hybrid ----    [combination of rolap and molap][storing in rolap -> fetching/analysis using molap by making cubes ]
